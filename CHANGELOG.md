# Changelog

All notable changes to the Missed Call Revenue Loss Calculator.

## [1.1.0] - 2026-07-05
### Added
- Open Graph and Twitter Card meta tags plus a meta description, so the link previews well when shared on social media or forums.
- A phone-emoji favicon.
- Accessibility improvements: proper form submit handling, an aria-live region on the results card, visible focus outlines on inputs/buttons, and a screen-reader status message when the summary is copied.

## [1.0.0] - 2026-07-05
### Added
- Initial public release: a free, no-signup calculator that estimates monthly and annual revenue lost to missed calls, plus payback period and ROI on a missed-call solution.
- Industry presets (home services, legal, dental, real estate, auto repair, restaurant).
- Copy-to-clipboard summary button.
- Published live via GitHub Pages.

### Fixed
- Copy-to-clipboard button silently failing in some browser contexts; added a document.execCommand fallback.
